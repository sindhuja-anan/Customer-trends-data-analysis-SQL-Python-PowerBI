# Customer Trends Data Analysis (SQL | Python | Power BI)

## 📌 Business Problem
Analyze customer shopping behavior to identify revenue drivers, customer segments, and purchasing trends to improve business decisions.

## 🛠 Tools Used
- SQL (Data Extraction & Queries)
- Python (Data Cleaning & EDA - Pandas, Matplotlib, Seaborn)
- Power BI (Interactive Dashboard Creation)

## 📊 Project Workflow
1. Data Cleaning using Python
2. Data Analysis using SQL Queries
3. Insight Generation using EDA
4. Dashboard Creation in Power BI

## 🔍 Key Insights
- Young adults contribute higher sales
- Subscription customers show higher spending
- Certain product categories drive maximum revenue
- Repeat customers generate stable profit

## 📈 Dashboard
Power BI dashboard created with KPIs:
- Revenue by Category
- Age Group Analysis
- Subscription Impact
- Product Performance
